Projeto HugMe

Será um projeto de uma pequena loja que vende pelúcias.
Terá um estoque e para as pelúcias finalzadas. Dependendo do tipo de pelucia e o material, seus preços serão diferenciados.  
Terá cadastro de clientes.
Intenção de fazer algumas classes que lidem com a composição dos bichinhos. Possívelmente adicionarei algo para materiais exóticos e mais algumas complexidades 

Ursos comuns têm uma clausula de doação para instituições que cuidam de crianças carentes.

Os clientes entarão numa tela principal, onde escolherão o tipo de pelucia que estes querem. 
Algumas pelúcias especifícas têm mais ou menos detalhes. Ao fazer a compra, o usuário fará um cadastro simples, e a pelúcia vai entrar no seu inventário e ser "linkada" com a conta do usuário
Pelucias que contêm doações terão um countdown no perfil do usuário 
Após a compra realizada, a pelúcia irá informar o usuário seu nome.
